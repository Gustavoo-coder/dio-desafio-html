O que é HTML?

HTML (Hiper Text Markup Language) é a linguagem mãe para estruturar um site. Estruturar é definir o que vai conter em nossa pagina, HTML segundo a W3schools é a linguagem de marcação padrão para criação de páginas da web

Estrutura Basica do HTML:

A estrutura basica do HTML Consiste em configurações pré-definidas no documento HTML, o que é o que chamamos de estrutura básica. Veja a tabela abaixo:

Estrutura Basica Significado
!DOCTYPE html Indica para o navegador que o site é feito em html
html tag que aborda tudo nosso documento html
head (cabeça) Tag em que definimos as configurações principais do nosso site
body (corpo) Lugar onde tudo o conteudo da nossa pagina vai aparecer para o usuario

Entendendo HTML semantico

Semantica com base no dicionário é o estudo do significado das palavras, frases e expressões, tanto em relação à sua forma (significante) quanto ao seu conteúdo e contexto. Basicamente no HTML há tags que possuem significado, e é de extrema importãncia que devemos utilizar essas tags, pois elas possuem grandes significado para os navegadores e principalmente pela acessibilidade web.
